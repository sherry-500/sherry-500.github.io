XGBoost: Principles, Engineering Implementation

sherry-500
2025-08-06
Machine Learning
Machine Learning

传统的机器学习模型，比如决策树，随机森林等通常难以在复杂数据集上获得高准确率，XGBoost（eXtreme Gradient Boosting）极致梯度提升是梯度提升的一种工程实现，具备高效，快速，高可扩展性的特点。

  • XGBoost使用决策树作为基学习器，顺序地结合多个基学习器以提高模型表现，每棵树被训练来纠正上一棵树的错误。
  • XGBoost具有内置的并行处理能力，能够在大规模数据集上快速地训练。它也支持用户自定义地调整模型参数，以实现特定问题上的性能优化。

数学原理

给定一个包含n个样本，m个特征的数据集𝒟 = {(xi, yi)} (|𝒟| = n, xi ∈ ℝm, yi ∈ ℝ)，假设XGBoost模型由K棵树组成，输出是这K棵树的线性组合。 其中ℱ = {f(x) = wq(x)}(q : ℝm → T, w ∈ ℝT)，q用于表示树的结构，将样本映射到对应的叶子节点的索引，T是一棵树中叶子节点的总数，w是叶子节点权重。与决策树不同的是，每棵回归树在每个叶子节点上都包含一个连续型的分数，wi表述第i个叶子的分数。

XGBoost实现函数空间内的加性优化，对于每个样本，我们使用q给定的树的决策规则将其分类到叶子节点，通过计算每棵树中对应的叶子节点的分数的和作为最终的预测。

正则化的学习目标

XGBoost的目标函数由两部分组成：损失函数和正则化项。损失函数衡量模型拟合数据的效果，正则化项控制树的复杂度。 XGBoost通过加入显示的正则化项控制树中叶子的数量和得分的大小，防止过拟合。

分裂查找算法

通常遍历所有可能的树结构q是不可能的，可以从一个单一的叶子节点开始，迭代地向树中增加分支。假设IL，IR分别是是分裂后的左子节点和右子节点的样本集合，令I = IL ∪ IR，则分裂后的损失减少量 如果ℒsplit为正时，则有必要进行分裂，否则不需要分裂。我们也可以使用ℒsplit评价分裂点的好坏，ℒsplit越大，分裂点越好。

精确贪心算法

为了找到最佳分裂点，精确贪心算法遍历所有特征的所有可能的分裂点。遍历连续型特征的所有可能分裂点对计算要求很高，为了高效地运行，这种算法必须先按照特征值的大小对数据进行排序，然后按序访问数据，积累计算ℒsplit所需的梯度数据。

近似算法

精确贪心算法不适用于数据不能完整加载进内存和分布式的场景，近似算法针对这一局限性给出了近似最优解。近似算法线根据特征分布的分位数提出候选分裂点，然后将连续型特征映射到由这些候选点划分的桶中，汇总每个桶内的梯度数据，并基于汇总的梯度数据在候选点中选出最佳分裂点。

该算法有两个变种，差别在于候选点给出的时间。

  • global variant：在树构建的初始阶段给出所有的候选点，在任意的树的层次使用相同的候选点做分裂查找
  • local variant：在每次分裂后提出新的候选点

global方法提出候选分裂点的步骤比local方法少，但通常需要更多的候选分裂点。local方法在分裂后重新提出候选分裂点，因此可能更适用于构建更深的树。实验证明当候选分裂点足够多时，global方法能和local方法一样准确。

稀疏感知

在实际应用中，输入x很可能是稀疏的。这种稀疏性由多种原因造成：1）数据中存在缺失值；2）经常出现零值；3）特征工程的产物，如one-hot编码。为了使算法能够感知数据中的稀疏模式，XGBoost提出在每个非叶子节点中增加一个默认方向。当输入在分裂的特征上缺失时，样本被分到默认方向。

最佳的默认方向是从数据中学到的，分别考虑将缺失值分入左分支和右分支的分裂增益，增益更大的即为默认方向。

收缩和列采样

为了进一步控制过拟合，XGBoost还采用了收缩和过拟合的技术。收缩在每次提升树的步骤之后，通过一个因子η缩放新添加的权重。类似于梯度下降法中的学习率，收缩减少了每棵树的影响，并为未来的树流程改进模型的空间。列采样技术在随机森林中也被用到，防止过拟合的同时也加速了并行算法的计算。

工程实现

列块并行学习

在树生成的过程中最耗时的一个步骤就是每次寻找最佳分裂点时都需要对特征的值进行排序。XGBoost在训练之前会根据特征对数据进行排序，然后保存在块结构中，每个块结构采用稀疏矩阵存储格式。后面的训练过程可以重复使用块结构，能大大减少计算量。

具体的，在寻找最佳分裂点时，我们可以顺序地遍历预排序结果，收集性地查找所有叶子节点的最佳分裂点。因此，与决策树采用深度优先的生长策略不同，XGBoost采用广度优先的生长策略。此外，分块存储后的多个特征之间互不干涉，可以使用多线程同时对不同的特征进行分裂点查找，即特征的并行化处理。

令d为树的最大深度，K为树的总数，∥x∥0为训练数据的矩阵中非零值的数量。精确贪心算法的时间复杂度为O(Kd∥x∥0log n)，如果使用块结构，时间复杂度仅为O(Kd∥x∥0+∥x∥0log n)。其中O(∥x∥0log n)是一次性排序的成本。
