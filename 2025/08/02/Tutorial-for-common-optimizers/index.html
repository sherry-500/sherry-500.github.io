Tutorial for common optimizers

sherry-500
2025-08-02
Deep Learning
Deep Learning, Machine Learning, Optimization methods

深度学习模型的优化过程可以类比为在山地中走到海拔最低点的过程，地势的起伏代表损失函数值的变化，不同的位置代表不同的参数取值，总体的最低点代表模型的最佳参数，每走一步可以理解为调整一次模型的参数。

优化器提供了下山的策略，主要关注两个方面：

  • 每一步向哪个方向走
  • 每一步走多大的步子

公式和定义：

  1. 待优化参数θ
  2. 目标函数J(θ)
  3. 学习率η

GD（gradient descent）

梯度下降法是许多优化算法构建的基础。

更新公式 GD的关键特征：

  • 每一步都使用全部的数据集
  • 结果一致，但可能较慢
  • 可能陷入局部最小值
  • 对学习率敏感，如果步长太大，会导致模型在损失函数的最小值附近来回震荡，无法收敛；步长太小，模型的学习过程会非常缓慢，甚至陷入局部最优解

SGD（stochastic gradient descent）

每次迭代只使用一小批数据（mini-batch）来更新参数

更新公式 优点

  • 每次只使用一小批样本更新模型参数，训练速度快，内存占用小
  • 批数据的随机性有利于优化的方向跳过局部最小值点

缺点

  • 对学习率敏感，学习率通常需要随时间逐渐减小，避免错过全局最低点
  • 训练初期，震荡明显，收敛速度较慢
  • 陷入局部最优点时，梯度为0，无法继续更新参数

SGD + Momentum（动量法）

在SGD的基础上引入动量，在梯度下降的过程中加入惯性，加速收敛，减少震荡。 一阶动量是各个时刻梯度方向的指数移动平均值，每次的下降方向主要是此前累积的下降方向，并略微偏向当前时刻的下降方向。

优点

  • 缓解了SGD在局部最优点梯度为0，无法持续更新的问题，动量加持有助于从局部最优点跳出
  • 可以减少震荡，加速收敛

缺点

  • 没有完全解决陷入局部最优点的问题，当局部沟壑较深，动量也用完了，依然会在局部最优来回震荡

AdaGrad

AdaGrad引入了二阶动量，对每个参数采用自适应的学习率。SGD及其变种以同样的学习率更新每个参数，但深度神经网络往往包含大量的参数，这些参数并不是总会被用到（如大规模的embedding）。对于经常更新的参数，我们已经累积了大量关于它的知识，不希望被少量样本影响太大，写昂学习率慢一点；对于偶尔更新的参数，我们了解的信息太少，希望从每个偶然出现的样本上多学一些，学习率大一点。 优点

  • 在稀疏数据场景下表现好
  • 为每个参数采用自适应学习率

缺点

  • 二阶动量单调递增，会使得学习率单调递减至0，可能使学习过程提前结束
  • 可能错过全局最优解

RMSProp

改变了二阶动量的计算方法，计算梯度平方的指数移动平均值，避免学习率快速减小 优点

  • 避免了二阶动量持续累积，导致训练过程提前结束

Adam （AdaptIve Moment Estimation）

结合了Momentum和RMSprop，分别估计一阶和二阶矩 优点

  • 使用偏差校正（bias correction），矫正初始m0 = V0 = 0带来的影响
  • 自适应学习率
  • 超参数鲁棒性好，相比于其他优化器对超参数的选择不太敏感
  • 收敛速度快
  • 防止梯度的震荡和在鞍点的静止

缺点

  • 自适应学习率算法可能会对前期出现的特征过拟合，后期才出现的特征很难纠正前期的拟合效果。可能错过全局最优解

AdamW

AdamW将梯度更新和权重衰减分离，在完成Adam的自适应梯度更新后，额外执行一次与梯度无关的线性衰减。Adam将L2正则项直接加在损失函数中，导致在后续的一阶，二阶矩估计中都包含了正则化项，在自适应学习率优化器中，这种操作会导致权重衰减强度被学习率缩放。AdamW将权重衰减从梯度更新公式中解耦，单独施加到参数更新步骤。使得动量和二阶矩估计仅反映原始梯度信息，不受正则化干扰。
