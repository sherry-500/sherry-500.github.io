basic definitions of Information Theory

sherry-500
2025-06-09
Information Theory › Statistics
Information Theory, Statistics

之前做特征选择的时候用到过互信息，顺便把相关的信息论的基本概念总结一下

信息量

信息量是对某个事件发生或者变量出现的概率度量，一个事件发生的概率越低，所包含的信息量越大。并且信息量应该满足I(XY) = I(X) + I(Y)，X, Y是两个独立事件，香农提出了一个衡量信息量的公式： I = −log p

熵

熵衡量一个系统的稳定程度，其实就是一个系统中所有变量的信息量的期望。

H(X) = −∑x ∈ XP(X)log P(X) = −E[log P(X)]

  • 事件发生的不确定性越高，系统越不稳定，熵就越高

  • 对于连续变量，熵可以理解为它的概率密度函数 H(X) = ∫−P(X)log P(X)

  • 联合熵：多个联合变量的熵，也就是将熵的定义推广到多变量的范围

    H(X, Y) = −∑x ∈ X∑y ∈ YP(XY)log P(XY) = −E[log P(XY)]

  • 条件熵：一个随机变量给定的情况下，系统的熵 显然可得： H(X, Y) = H(X) + H(Y|X) = H(Y) + H(X|Y) 证明：

  • 相对熵（KL散度）：

    相对熵衡量的用一个分布q近似另一个基准分布p时的信息损失量。

    1. 相对熵是一种非对称度量两个概率分布的差异的指标
    2. 在机器学习中，不是交叉熵越接近0越好，而是相对熵越接近0越好

  • 交叉熵：也是用来衡量两个概率分布之间的差异性 HCE(p, q) = −∑ip(xi)log q(xi) 交叉熵是相对熵的一部分，因为在通常情况下p是已知的，H(P)是常量，此时交叉熵和相对熵是线性关系

    1. 当两个分布完全一致时，HCE = H(P)，取得最小值
    2. 两个分布的差异越大，交叉熵越大
    3. 交叉熵描述两个分布之间的接近程度是不具备对称性的

互信息

互信息衡量两个随机变量之间的依赖关系，它表示“已知一个变量的信息后，另一个变量的不确定性减少了多少”。

  1. 数学本质：基于概率分布的比较，计算联合分布P(x, y)与独立分布P(x)P(y)的差异。互信息越小，说明联合分布P(x, y)越接近独立分布P(x)P(y)，X, Y的相关性越弱。
  2. 对称性：I(X; Y) = I(Y; X)
  3. 与相关性不同：互信息量能能捕捉线性和非线性关系

从公式上看，互信息与信息增益很相似，都表现为确定一个变量后，另一个变量的不确定性减少了多少。不同在于：

  1. 互信息中两个随机变量的地位是相同的，信息增益中把确定的随机变量看成减少另一个随机变量不确定性是手段
  2. 互信息中Y是事件，信息增益中Y是分类方式
