How to select features

sherry-500
2025-06-09
Machine Learning
Feature Engineer, Machine Learning

特征筛选在机器学习中具有重要的意义，当数据中包含无关或冗余特征时，可能会使模型在训练中过拟合。特征筛选使模型更专注于与目标变量真正相关的特征，更准确地捕捉数据中的关键模式和关系，提高在新数据上的泛化能力和预测的准确性。同时减少计算资源的浪费，加快模型的训练速度。

本文将使用一个Kaggle上一个保险金预测数据集进行演示。

  • 类别特征：‘Gender’, ‘Marital Status’, ‘Education Level’, ‘Occupation’, ‘Location’, ‘Policy Type’, ‘Customer Feedback’, ‘Smoking Status’, ‘Exercise Frequency’, ‘Property Type’
  • 连续特征：‘Vehicle Age’, ‘Number of Dependents’, ‘id’, ‘Credit Score’, ‘Health Score’, ‘Previous Claims’, ‘Age’, ‘Insurance Duration’, ‘Annual Income’, ‘Policy Start Date’

基于统计量的特征筛选

可以统计出特征自身的分散程度或者特征与预测目标之间的相关程度，判断是否保留特征。接下来是常用的评估特征重要性的指标。

方差

方差衡量的是一个随机变量取值的分散程度，如果一个随机变量的方差很小，那么就很难对输出有什么影响。在进行特征选择时，可以丢弃方差特别小的特征。

需要注意的是，只有当各个特征的分布较为接近时，才能使用方差来衡量信息量。

Pearson相关系数

Pearson相关系数是两个随机变量的协方差和它们标准差的乘积的商，其中协方差描述了两个变量同时偏离其均值的程度，除以标准差之积起到了标准化的作用： 从几何意义上解释，Pearson相关系数可以看作两个随机变量经过标准化后在向量空间中的余弦相似度： Pearson相关系数反应了两个变脸之间的线性相关程度，取值范围在[−1, 1]之间，j绝对值越大，相关性越大，越接近0，相关性越小。Pearson相关系数适用于：

  • 两个变量都是连续数据
  • 两个变量的总体是正态分布，或者接近正态的单峰分布，否则其结果是不可靠的
  • 假设两个变量之间的关系是线性的
  • 没有异常数据，极端值会显著影响Pearson相关系数

互信息

互信息能够捕捉非线性关系，实际上是使用两个随机变量的联合分布逼近独立分布产生的KL散度。

互信息主要存在的问题是需要先将连续变量离散化，计算结果对离散化的方式敏感
