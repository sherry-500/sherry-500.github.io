Some Common Probability Distributions

sherry-500
2025-07-15
Probability Theory
Probability Theory

概率分布（probability distribution）是概率论中刻画随机变量取某些值以及以何种概率取值的概念，按照随机变量的种类，概率分布可以分为离散型概率分布和连续型概率分布以及奇异型概率分布。每个概率分布都有自己确定的分布函数，这些分布参数中可能有某些参数，一个概率分布代表了一大类有相似特征的随机变量。

离散型概率分布

离散型概率分布只会在可列个值处有概率，常见的离散概率分布有二项分布，泊松分布和几何分布。这些分布都有自己的概率质量函数（PMF），它给出每个可能结果的概率。

二项分布

二项分布是在伯努利试验的基础上进行的。

伯努利试验是在随机试验的条件下，结果只有两种可能：发生或者不发生的一种试验，并且每次试验的结果之间相互独立，互不干扰。

二项分布的定义为在重复n次的独立的伯努利试验中，探究成功的次数k。当试验次数为1时，二项分布服从0-1分布。

假设随机变量X服从参数为n（试验次数）和p（每次试验成功的概率）的二项分布，即X ∼ Binomial(n, p)

多项分布

多项分布是二项分布的推广，其中随机试验的积结果是k中互斥的离散状态，每种状态出现的概率为pi, p1 + … + pk = 1，在这个前提下进行n次实验，用xi表示每种状态出现的次数，x1 + … + xk = n，记作X ∼ PN(n, p1, …, pk)。现实中有很多符合多项分布的例子，比如扔骰子。

多项分布的特点如下：

  • 每次试验有多种可能的结果
  • 每种结果有各自发生的概率，所有结果的发生概率之和为1
  • 歌词试验相互独立，互不干扰

泊松分布

泊松分布用于描述单位时间内随机事件发生的次数的概率分布。泊松分布可以被理解为伯努利试验次数趋向于无穷大，伯努利试验成功的概率趋向于无穷小时的二项分布。

随机变量X服从参数为λ的泊松分布，记作X ∼ π(λ)，我们可以从二项分布的PMF推导出泊松分布的PMF 泊松分布的唯一参数是速率λ（X的期望值），尽管泊松分布常用来对小概率事件建模，但λ可以是任意数字，并不总是很小。泊松分布总是向右倾斜，因为受到零发生障碍的抑制（不可能发生负数次事件），而在另一边是无限的。随着λ变大，泊松分布越像正态分布。

泊松分布需要满足两个假设条件：

  • 每单位时间的事件平均速率是恒定的
  • 事件是独立的

泊松分布的期望和方差都等于λ

连续型概率分布

当随机变量在给定范围内可以取无限多个值时，连续概率分布使用。连续分布的特征是概率密度函数（PDF），它描述随机变量落入特定区间的可能性。PDF曲线下的总面积等于1。连续型概率分布无法给出某一个数值的概率，我们更关心的是一个特定范围的概率。常见的连续概率分布有正态分布。

正态分布

X服从参数为μ, σ的正态分布，记作X ∼ N(μ, σ2)。正态分布是很多应用中国的合理选择。如果某个随机变量取值范围是实数，且对它的概率分布一无所知，通常会假设服从正态分布，这样做有两个原因：

  • 建模的任务的真实分布通常都接近正态分布，中心极限定理表明，多个随机变量的和近似正态分布。
  • 在具有相同方差的所有可能的概率分布中，正态分布的熵最大
