Direct Preference Optimization

sherry-500
2025-07-11
Machine Learning
LLM, Machine Learning

大规模无监督语言模型虽然可以学习广泛的世界知识和一些推理技能，但由于训练的无监督性它们的行为是难以控制的。目前获得可控性的方法是先收集人类对模型生成内容的相对质量的标签，然后使用通过人类反馈的强化学习（RLHF）对无监督的LM进行微调，使其与人类偏好一致。然而RLHF是一个复杂且不稳定的过程，需要先拟合一个反映人类偏好的奖励模型，再使用强化学习微调LM，最大化估计奖励，同时避免与原始模型偏离太远。DPO[^1]作为一种新的奖励模型参数化方法，能够在RLHF中封闭形式提取出相对应的最优策略，从而允许我们仅通过简单的分类损失来解决标准的RLHF问题。

DPO稳定，高效，计算成本轻量，无需拟合奖励模型，在微调期间从LM中采样，或执行显著的超参数调整。实验结果表明，DPO进行微调超过了基于PPO的RLHF的效果，更易于实现和训练。

RLHF

RLHF通常包含三个阶段：1. 监督微调，2. 偏好采样和奖励学习，3. 强化学习优化策略。

1. SFT

RLHF通常从在一个下游任务的高质量数据上通过监督学习微调一个预训练LM开始，获得一个模型。

2. 奖励建模

在第二个阶段，经过监督微调的模型为提示词生成答案对。这些答案会由人类标准者标注出更偏好的答案，记作，表示更优的答案。假设这些偏好是由某个潜在的奖励模型$r^{} (y|x)$
p^(y_1 \succ y_2 |x) = \frac{\text{exp} (r^(x, y_1))} {\text{exp} (r^(x, y_1)) + \text{exp} (r^(x, y_2))} = \sigma(r^(x, y_1) - r^*(x, y_2))
$$
假设有一个静态数据集$\mathcal{D} = \big{ x^{(i)}, y_w^{(i)}, y_l^{(i)} \big}{i=1}^Nr{\phi}(y, x)r_{\phi}(y, x)$的训练目标：

$$
\mathcal{L}R(r{\phi}, \mathcal{D}) = -\mathbb{E}{(x, y_w, y_l) \sim D}[\log{\sigma(r{\phi}(x, y_w) - r_{\phi}(x, y_l))}]
$$

对于LM来说，通常被初始化为SFT模型在最后一层transformer层后添加一个线性层，以获得奖励值的标量预测。

3. RL微调

在强化学习阶段，奖励模型为LM提供反馈。优化目标为最大化奖励和最小化与参考策略的KL散度
$$
\max_{\pi_{\theta} } \mathbb{E}{x \sim D, y \sim \pi{\theta}(y|x)}[r_{\phi}(x, y)] - \beta \mathbb{D}{\text{KL}} [\pi{\theta}(y|x) \text{ }|| \text{ } \pi_{\text{ref} }(y |x)] =
\max_{\pi_{\theta} } \mathbb{E}{x \sim D, y \sim \pi{\theta}(y|x)}[r_{\phi}(x, y) - \beta(\log \frac{\pi_{\theta}(y|x)} {\pi_{\text{ref}}(y|x)})]
$$
在实际应用中，参考策略模型被初始化为，策略模型也被初始化为。KL散度惩罚项用来防止模型过于偏离奖励模型所基于的分布，同时保持生成的多样性并防止模式崩溃到单一高奖励答案。由于language generation的离散特性，上述优化目标是不可微的，通常使用强化学习优化。

增加KL散度约束后，奖励更新为：

DPO

相比于RLHF使用偏好数据训练奖励模型，再依据奖励优化策略模型，DPO绕过了奖励模型这一步，直接依据偏好数据优化策略模型。DPO的核心思想是：利用奖励函数到最佳策略之间的解析映射关系，将关于奖励函数的损失函数转换为关于策略的损失函数。这种变量替换方法避免了拟合显式的独立奖励模型，同时仍然可以再现有的人类偏好模型下（如BT模型）进行优化。本质上，策略模型既代表了语言模型，也代表了（隐式）的奖励模型。

推导优化目标

对于任何奖励函数，参考模型和一般的无参策略类型，定义优化目标：
$$
\begin{equation}
\begin{split}
\max_{\pi} \mathbb{E}{x\sim \mathcal{D}, y \sim \pi} [r(x, y)] - \beta \mathbb{D}{\text{KL} }[\pi(y|x) \text{ } || \text{ } \pi_{ref}(y|x)] \
= \max_{\pi} \mathbb{E}{x \sim \mathcal{D} } \mathbb{E}{y \sim \pi(y|x)} [r(x, y) - \beta \log \frac{\pi(y|x)} {\pi_{ref}(y|x)}] \
= \min_{\pi} \mathbb{E}{x \sim \mathcal{D} } \mathbb{E}{y \sim \pi(y|x)} [ \log \frac{\pi(y|x)} {\pi_{ref}(y|x)} -\frac{1} {\beta}r(x, y)] \
= \min_{\pi} \mathbb{E}{x \sim \mathcal{D} } \mathbb{E}{y \sim \pi(y|x)} [\log \frac{\pi(y|x)} {\pi_{ref}(y|x) \exp(\frac{1} {\beta}r(x, y))}] \
= \min_{\pi} \mathbb{E}{x \sim \mathcal{D} } \mathbb{E}{y \sim \pi(y|x)} [\log \frac{\pi(y|x)} {\frac{1} {Z(x)}\pi_{ref}(y|x) \exp(\frac{1} {\beta}r(x, y))} - \log(Z(x))]
\end{split}
\end{equation}

Z(x) = \sum_y \pi_{ref}(y|x) \exp \left(\frac{1} {\beta} r(x, y)\right)

\pi^(y|x) = \frac{1} {Z(x)}\pi_{ref}(y|x) \exp \left(\frac{1} {\beta}r(x, y)\right)
\tag{1}
$$
因为对于所有的都有$\pi^(y|x) \ge 0\sum_y \pi^(y|x) =1\pi^(y|x)$
\begin{equation}
\begin{split}
\min_{\pi} \mathbb{E}{x \sim \mathcal{D} } \mathbb{E}{y \sim \pi(y|x)} [\log \frac{\pi(y|x)} {\frac{1} {Z(x)}\pi_{ref}(y|x) \exp(\frac{1} {\beta}r(x, y))} - \log(Z(x))] \
= \min_{\pi} \mathbb{E}{x \sim \mathcal{D} } [\mathbb{E}{y \sim \pi(y|x)} [\log \frac{\pi(y|x)} {\pi^(y|x)}] - \log(Z(x))] \
= \min_{\pi} \mathbb{E}{x \sim \mathcal{D} } [\mathbb{D}{\text{KL} } (\pi(y|x) \text{ } || \text{ } \pi^(y|x)) - \log Z(x)]
\end{split}
\end{equation}

\pi(y|x) = \pi^(y|x) = \frac{1} {Z(x)}\pi_{ref}(y|x) \exp \left(\frac{1} {\beta}r(x, y)\right)
\tag{2}

r(x, y) = \beta \log \frac{\pi(y|x)} {\pi_{ref}(y|x)} + \beta \log Z(x)
$$
我们使用$r^\pi^$
\begin{equation}
\begin{split}
\min_{r^}\mathcal{L}{DPO}(r^*, \mathcal{D}) &= \min{r^}-\mathbb{E}_{(x, y_w, y_l) \sim \mathcal{D} }[\log{\sigma(r^(x, y_w) - r^*(x, y_l))}] \
&= \min_{\pi^{\theta}} -\mathbb{E}{(x, y_w, y_l) \sim \mathcal{D} } \left[ \log \sigma \left( \beta \log \frac{\pi{\theta} (y_w | x)} {\pi_{\text{ref} }(y_w|x)} - \beta \log \frac{\pi_{\theta}(y_l | x)} {\pi_{\text{ref} } (y_l | x)}\right) \right]
\end{split}
\end{equation}
$$
该目标增加了对偏好数据的可能性，并减少了非偏好数据的可能性。

梯度分析

为了从原理上理解DPO，分析损失函数的梯度：

$$
\nabla_{\theta} \mathcal{L}{\text{DPO} }(\pi{\theta}, \pi_{\text{ref} }) = -\beta \mathbb{E}{(x, y_w, y_l) \sim \mathcal{D} } \left[ \sigma(\hat{r}{\theta} (x, y_l) - \hat{r}{\theta} (x, y_w)) \text{ } \left[ \nabla{\theta} \log \pi(y_w | x) - \nabla_{\theta} \log \pi (y_l | x) \right] \right]
$$

其中$\hat{r}{\theta} (x, y) = \beta \log \frac{\pi{\theta} (y | x)} {\pi_{\text{ref} } (y | x)}\pi_{\theta}\pi_{\text{ref} }y_wy_l$的可能性。更重要的是，样本的权重是通过隐式奖励模型对非偏好的评分比对偏好的评分高多少来衡量的，即隐式奖励模型认为策略模型错误的程度。

基本流程

DPO的基本流程可以分为两步：

  1. 为每个提示词采样补全，标注人类偏好，构建离线数据集。
  2. 对于给定的，通过最小化$\mathcal{L}{\text{DPO} }\pi{\theta}$。

Conclusion

DPO提供了一种无需强化学习的语言模型学习偏好数据的范式，DPO并未将偏好学习问题强行纳入标准强化学习框架以使用现成的RL算法，而是找到了语言模型策略和奖励函数之间的映射关系，从而能直接用简单的交叉熵损失训练模型以满足人类偏好，无需强化学习，也不损失通用性。DPO 几乎无需调整超参数，其性能就与现有的基于 PPO 的 RLHF 算法相当甚至更优，因此显著降低了从人类偏好中训练语言模型的门槛。
